# tech-notes

My Tech Notes

## Prerequisites

- Nodejs 18+
- VitePress

## How to?

```bash
# Clone the repository
$ git clone https://github.com/chensoul/tech-notes.git

# install dependences
npm install

##local run
npm run docs:dev

# build dist
npm run docs:build
```

Visit: http://localhost:5173/tech-notes/
