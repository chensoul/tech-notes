## Installation

Easy way to install Java is using [SDKMAN](https://sdkman.io/)

```bash
$ curl -s "https://get.sdkman.io" | bash
$ source "$HOME/.sdkman/bin/sdkman-init.sh"
$ sdk version
$ sdk list java
$ sdk install java 21.0.5-tem
$ sdk install java 8.0.432-zulu
```
